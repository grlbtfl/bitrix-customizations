Политика конфиденциальности

Расширение Bitrix24 Sound Customizations не собирает, не передаёт и не обрабатывает персональные данные пользователей.

Все настройки расширения (например, ссылка на пользовательский звуковой файл) хранятся локально в браузере пользователя с использованием стандартного API хранилища браузера и не передаются на какие-либо серверы.

Расширение не использует аналитику, не отслеживает действия пользователя, не получает доступ к аккаунтам, переписке или иной личной информации.

Разработчик расширения не имеет доступа к каким-либо данным пользователя.

Если у вас есть вопросы, вы можете связаться с разработчиком через страницу проекта на GitHub.

Privacy Policy

The Bitrix24 Sound Customizations extension does not collect, transmit, or process any personal user data.

All extension settings (such as a custom sound file URL) are stored locally in the user's browser using the standard browser storage API and are not sent to any external servers.

The extension does not use analytics, does not track user activity, and does not access user accounts, messages, or any other personal information.

The developer does not have access to any user data.

If you have any questions, you may contact the developer via the GitHub project page.
